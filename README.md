# Rlicker
A fast and lightweight autoclicker in rust released as AppImage so that it is compatible across distros and easy to use

To build from source download repository as zip, unzip and run cargo build --release this builds the executable at auto/target/release in the folder
###### If u want an AppImage run cargo AppImage after installing it with cargo install cargo-appimage this will make the AppImage which is located in /target/Rlicker.AppDir if u want an icon make sure to put Rlicker.png inside /auto/target/Rlicker.AppDir
